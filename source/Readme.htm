Manual Install

To install, you can xcopy/ftp deploy to IIS7 server running ASP.NET 3.5 SP1 with ASP.NET MVC 1.0.

If your server has ASP.NET MVC 1.0 installed, you do not need to copy the System.Web.Mvc.dll file to your server.

You must give modify access to the App_Data folder and to the Service.config and Users.config files.
